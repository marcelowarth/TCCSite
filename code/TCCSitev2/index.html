BEM VINDO!

Este Site e Projeto foi desenvolvido como Trabalho de Conclusão do curso de Bacharelado em Ciências da Computação da Universidade Feevale.

O projeto tem como objetivo ensinar história da computação e lógica de programação utilizando a história como motivador e blocos de comando para realizar os desafios.



AVISOS!

Este projeto ainda está em construção e pode conter problemas.

Por favor, utilize o navegador Firefox, o navegador Google Chrome não permite executar o plugin necessário para a utilização correta do projeto.



Instruções

1. Acesse a página Projeto

2. Aguarde a tela inicial do projeto iniciar.

3. Navegue, aprenda e tente realizar os desafios.

4. Por favor, quando terminar, responda o questionário que existe na página Pesquisa.

5. Opcionalmente, crie seu próprio desafio na página Crie.
