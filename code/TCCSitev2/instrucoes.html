Instruções Gerais

Quando um desafio for selecionado, aparecerá a tela de resolução do desafio, esta tela, sempre inicia com uma caixa de diálogo na parte superior da mesma. Se a caixa sumir, clique no auto-falante para que a mesma apareça novamente.

Para adicionar comandos para o personagem seguir, arraste o comando da lista na esquerda para a área verde a direita. E para remover só arraste para fora da área verde. Os comandos podem ser reorganizados arrastando-os dentro da área verde.

Pegar Entrada

O comando Pegar Entada, quando executado irá até a esteira superior e pegará o primeiro item da mesma, removendo da esteira e segurando sobre a cabeça.

Levar á Saída

O comando Levar á Saída, quando executado irá levar a esteira inferior o item que o personagem estiver segurando, removendo o mesmo das mãos do personagem.

Pular para

O comando Pular para serve para que o usuário possa criar loops de comando ou pular sequências de comando. Quando adicionado, são criados 2 blocos, um indicando o momento em que será pulado e outro com uma letra indicadora para onde será pulado. No exemplos abaixo foi criado um loop, e no outro foi pulado uma sequência de comandos.

Copiar para

O comando Copiar para, serve para armazenar o valor que está sendo segurado pelo personagem em uma posição no centro da tela, será criada uma cópia do valor naquela posição e o personagem continuará com o item na cabeça. Quando o comando for adicionado na área verde, será necessário selecionar uma das 8 áreas destacadas em verde no centro da tela.

Copiar de

O comando Copiar de, tem o mesmo funcionamento do Copiar para, porém o que ele fará é buscar o item que existe na posição do centro da tela, isto não removerá o item ali presente, somente criará uma cópia sobre a cabeça do personagem.

Somar com

O comando Somar com é utilizado para somar o item que o personagem está segurando com o item de uma das posições do centro da tela, quando utilizado, o item que o personagem está segurando será substituido pelo item com novo valor.

Diminuir com

O comando Diminuir com é utilizado para diminuir o item que o personagem está segurando com o item de uma das posições do centro da tela, quando utilizado, o item que o personagem está segurando será substituido pelo item com novo valor.

Se Zero

O comando Se Zero possui o mesmo fucionamento do comando Pular para, porém ele somente irá pular para a posição designada se o item segurado pelo personagem for zero, ser não for, será executado o próximo comando da sequência. Abaixo um exemplo onde o é verificado se o item pego é zero e levado a saída, se não for volta para o início.

Se Negativo

O comando Se Negativo possui o mesmo fucionamento dos comandos Pular para e Se Zero, porém ele somente irá pular para a posição designada se o item segurado pelo personagem for negativo, ser não for, será executado o próximo comando da sequência.

Aumente

O comando Aumente serve para aumentar em 1 o valor de um item de uma das posições centrais da tela, quando o mesmo é adicionado na área verde, é necessário que seja selecionada a posição na área central.

Diminua

O comando Diminua serve para diminuir em 1 o valor de um item de uma das posições centrais da tela, quando o mesmo é adicionado na área verde, é necessário que seja selecionada a posição na área central.
